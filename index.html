Kurzanleitung für Canvas API

Wozu braucht es diese WEB-API

Canvas-API bietet eine Möglichkeit, Grafiken mithilfe von JavaScript und dem HTML Canvas-Element zu zeichnen. Sie kann unter anderem für Animationen, Spielgrafiken, Datenvisualisierung, Fotobearbeitung und Echtzeit-Videobearbeitung verwendet werden. Die Canvas-API konzentriert sich hauptsächlich auf 2D-Grafiken. Die WebGL-API, die ebenfalls das Canva-Element nutzt, ermöglicht das Zeichnen von hardwarebeschleunigten 2D- und 3D-Grafiken.

Codebeispiele


        <canvas id="meinCanvas" width="500" height="400" style="border:1px solid #000;"></canvas>
        <script>
            const canvas = document.getElementById("meinCanvas");
            const ctx = canvas.getContext("2d");
        
            // Rechteck zeichnen
            ctx.fillStyle = "blue";
            ctx.fillRect(50, 50, 150, 100);
        
            // Kreis zeichnen
            ctx.beginPath();
            ctx.arc(300, 150, 50, 0, Math.PI * 2);
            ctx.fillStyle = "red";
            ctx.fill();
            ctx.closePath();
        
            // Linie zeichnen
            ctx.beginPath();
            ctx.moveTo(50, 300);
            ctx.lineTo(400, 300);
            ctx.strokeStyle = "green";
            ctx.lineWidth = 5;
            ctx.stroke();
            ctx.closePath();
        </script>
